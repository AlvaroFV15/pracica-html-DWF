Definiciones Basicas

Base de Datos: Colección de datos organizados y relacionadoos entre si, que permiten la rapida selección de información. Ejempla: Productos, Pedidas.

Tipos de bases de datos

Relacional:La mas comun, basada en tablas (filas = registros, columnas = campos), Las datos pueden estar relacionados entre tablas a través de indices.

Distribuida: Los datos se almacenan en mültiples servidores y se intercambia información entre ellos.

Orientada a Objetos: Almacena información en forma de objetos, permitiendo modelar datos complejos.

Gráfica: Almacenan datos y sus relaciones, facilitando recorridos rápidos entre nodos conectados.

NoSQL: Na usan tablas sino colecciones de elementos heterogéneos.


Características de una base de datos

  • Compatibles con varios sistemas operativos y con múltiples accesos simultáneos. Permiten una gestión segura de usuarios. Flexibilidad al gestionar la información. Son independientes al código lo que permite que se pueda utilizar diversos lenguajes de información. Independencia de datos. Acceso por múltiples usuarios. Respaldo y recuperación.
